My current focus is AI agent workflows: how AI can analyze messy visual or text input, propose structured actions, and let humans review, accept, reject, and edit the results.

## Current Project

### Cinematic Comic Video Converter

A React/TypeScript browser tool that turns static comic pages into cinematic motion-video plans.

The project explores an AI director-assistant workflow:

* AI analyzes comic pages, panels, characters, dialogue, and visual details
* AI suggests camera motion, mood, BGM, and SFX ideas
* Users review, accept, reject, and edit suggestions
* Accepted suggestions become project data
* The tool previews and exports cinematic camera movement over the comic page
